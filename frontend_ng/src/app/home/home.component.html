Technologies Used:
  • Dockerized Angular 7 App with Angular Material in One Kubernetes Cluster. The Angular App has the following features
    • Authentication using OpenID Connect
    • Sign-in with Google - Used OAuth 2.0 option provided by Google API Console
    • Only Logged in users will be able to see the grid. It will load the data dynamically once the user scroll down.
  • Dockerized Java API (Spring Boot, JPA, Hibernate Restful CRUD API) on another Kubernetes Cluster
  • Continuous Integration & Continuous Delivery done using the following:
    • Jenkins for CI
    • Spinnaker for CD
  • Cloud SQL (around 1,00,000 records)
